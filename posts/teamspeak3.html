TeamSpeak3语音

TeamSpeaka开黑语音软件

TeamSpeak是全球流行的团队语音通讯工具软件，军用级安全性、无延迟的性能和无与伦比的可靠性

ts1.cn为盗版软件网站，请自行甄别。

使用

安装

打开下载的安装包进行安装。

  1. 点击Next
  2. 将侧边导航栏拉到最下面然后将下方勾选
  3. 选项1：安装为电脑所有人使用 选项2。仅为自己安装使用。
  4. 选择安装路径
  5. 选择要存储配置的位置 1： 建议保存在用户自己的文件中 2：保存在安装文件夹
  6. 是否安装Overwolf软件（这是一个插件，可不安装）

配置

  1. 打开软件会有一个声明，侧边导航栏拉到最下面点击i accept即可
  2. 随后会出现一个账号登录，ts可以直接关闭免登录使用。也可以点击create accout创建一个账号
  3. 随后点击connections-> connet连接服务器
  4. 输入ip地址和服务器密码

    进入服务器即可使用

汉化

软件默认是英文语言和语音，可安装插件实现汉化语言和语音。

  1. 进入服务器右键大厅点击Open File Browser
  2. 下载汉化文件（或者文章上面直接下载汉化插件）
  3. 下载完后开打文件，点击install选择yes
  4. 重启之后软件就是中文显示。（语音包同理）

快速进入服务器

如果自己只有一个服务器使用，那么手动点击连接会繁琐，接下来设置达到打开软件自动连接服务器的效果。

  1. 首先进入自己使用的服务器，点击上方的书签管理器
  2. 进入之后点击下方的进阶选项（这里是点击之后才显示的基本选项），将启动连接服务器勾选即可

    麦克风设置

    首先打开工具->设置->音频输入

启动麦克风方式：

  1. 按键通话：设置键位，按下启用麦克风，松开禁用麦克风
  2. 持续传输：自由麦，一直启动麦克风（不推荐）
  3. 语音检测：自由麦基础上添加了一个阈值，音量到达一定数值启用麦克风。（推荐）
    ts默认是感应麦。但是如果背景噪音比较大麦克风也会一直触发，比较吵，所以需要将麦克风触发阈值调高，达到屏蔽底噪效果。

语音检测模式：

  1. automatic：自适应设置感应值
  2. volume Gate：手动调节阈值
  3. Hybrid：无（暂时不确定和volume Gate区别。都是可以手动设置阈值）
    我们模式选择volume Gate，点击开始测试，说话看下自己的底噪，将值拉到底噪之上即可。

创建频道及设置


点击空白处或者当前频道右键创建频道

主要设置两项：
频道类型：频道存在时间
音质中品质：决定此频道玩家音质如何，建议拉高
如何品质拉最低，再好的麦克风，大家听到的都是炸麦音质

管理权限


点击权限->服务器组

1：设置管理员图标

2：管理员权限
3：添加权限
4：更改权限名称

server admin 不需要进行更改，此职位为服务器管理员

超管


权限范围： 全频道
踢出用户： 超管及以下
创建频道： 永久频道
可给予职位： 频道主及以下
访问权限： 全频道（无视频道密码）

频道主

权限范围： 当前频道
踢出用户： 频道内用户
创建频道： 子永久频道
可给予职位： 管理员及以下
访问权限： 当前频道所有子频道（无视频道密码）

管理员


权限范围：当前频道（或当前子频道）
踢出用户： 游客
创建频道： 子半永久频道
可给予职位： VIP
访问权限： 当前频道（或当前子频道）

开启高级权限


TEAMSPEAK 3 权限列表

注意事项

ts的服务器权限和管理权限都是本地存储，所以我们需要创建密钥以防数据丢失，权限全无

如果是服务器主丢失权限那么只能重新搭建服务器才可以恢复权限

权限密钥的创建只能服主创建

创建权限密钥

  1. 点击软件上方的权限（M），点击权限密钥
  2. 创建服务器密钥类型

    如果是服主，那么只需要创建服主密钥即可，这个权限最大后续权限可在自行分配

    多创建几个，每个密钥都是一次性的

密钥使用

点击上方权限（M）->使用权限密钥输入即可恢复权限。

账号密钥

当我们创建账号登录使用的时候，如果更换电脑登录那么软件需要我们输入账号的密钥，这个密码在创建的时候会提示。
如果我们丢失可以在工具->选项->myTeamSpeak->新的恢复密钥来创建新的密钥

创建新的密钥之后权限会丢失，请提前备份好权限密钥
